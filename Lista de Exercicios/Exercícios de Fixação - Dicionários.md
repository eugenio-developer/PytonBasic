# Exercícios de Fixação em Python

## Parte: Dicionários

**Objetivo:** Praticar o uso de dicionários para armazenar e manipular dados estruturados.

### Exercícios

1. Crie um dicionário que armazene os filmes favoritos de 3 pessoas.
2. Crie um dicionário onde cada chave seja o nome de um aluno e os valores sejam suas 3 notas.
3. Faça um dicionário com informações de clientes: nome, idade e lista de produtos comprados.
4. Crie um sistema que simule uma agenda de contatos onde cada pessoa pode ter mais de um telefone.
5. Crie um dicionário com cursos oferecidos por uma escola. Cada curso deve ter: nome, duração e lista de alunos inscritos.
6. Crie um dicionário com cidades e as respectivas temperaturas mínimas e máximas dos últimos 3 dias.
7. Crie um dicionário que represente uma conta bancária com chaves: titular, número da conta, saldo e lista de transações.
8. Crie um dicionário com nomes de esportes como chave e listas de atletas como valor.
9. Crie um dicionário onde cada produto tem preço, estoque e lista de avaliações de clientes.
10. Faça um dicionário com disciplinas e, para cada uma, armazene o professor responsável e os alunos matriculados.
11. Crie um dicionário com músicas, onde cada música tem artista, ano e gêneros (lista).
12. Simule um sistema de biblioteca com livros, onde cada livro tem título, autor e uma lista de datas de empréstimos.
13. Crie um dicionário com diferentes marcas de carros e, para cada marca, uma lista de modelos.
14. Crie um dicionário com equipes de futebol e, para cada uma, uma lista com os jogadores.
15. Faça um programa que leia o nome de 5 pessoas e, para cada uma, peça 2 hobbies e salve em um dicionário.
