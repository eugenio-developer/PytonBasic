# Exercícios de Fixação em Python

## Parte 3: Estruturas de Controle

**Objetivo:** Praticar o uso de estruturas condicionais (`if`, `elif`, `else`) e de repetição (`for`, `while`).

### Exercícios

1. Peça ao usuário um número e diga se ele é positivo, negativo ou zero.
2. Crie um programa que receba três números e diga qual é o maior deles.
3. Escreva um código que peça ao usuário sua idade e exiba se ele é criança (até 12), adolescente (13 a 17), adulto (18 a 59) ou idoso (60+).
4. Faça um programa que imprima os números de 1 a 10 usando `for`.
5. Faça o mesmo que o anterior, mas usando `while`.
6. Escreva um programa que leia 5 números e mostre a soma deles.
7. Crie um laço que peça números ao usuário até que ele digite 0. No final, exiba a soma total.
8. Escreva um código que conte quantos números pares existem entre 1 e 100.

##  Parte 4: Estruturas de Dados (Listas, Dicionários, Conjuntos e Tuplas)

**Objetivo:** Praticar o uso das principais estruturas de dados nativas do Python.

### Exercícios

1. Crie uma lista com 5 nomes e imprima todos os nomes com um `for`.
2. Adicione um novo nome à lista do exercício anterior e remova o primeiro nome.
3. Crie uma tupla com 4 números. Tente alterar um valor (e observe o que acontece).
4. Peça 3 notas ao usuário e armazene em uma lista. Depois, calcule a média.
5. Crie um dicionário com as chaves: `nome`, `idade`, `cidade`. Mostre os valores.
6. Adicione uma nova chave ao dicionário do exercício anterior: `profissão`.
7. Dado o dicionário `aluno = {"nome": "João", "notas": [7.5, 8.0, 6.5]}`, calcule a média das notas.
8. Crie um conjunto com 5 números, incluindo repetições, e mostre os valores únicos.
9. Escreva um programa que conte quantas vezes cada letra aparece na palavra "banana".
10. Peça ao usuário 5 números e armazene-os em uma lista. Depois, exiba:
    - O maior número
    - O menor número
    - A soma dos números
    - A lista ordenada
