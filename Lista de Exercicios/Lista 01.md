# Exercícios de Fixação em Python

## Parte 1: Tipos Primitivos em Python

**Objetivo:** Praticar o uso e manipulação dos tipos primitivos: `int`, `float`, `bool`, `str`.

### Exercícios

1. Declare uma variável do tipo `int` com qualquer valor e imprima seu tipo usando `type()`.
2. Crie duas variáveis, uma `int` e outra `float`. Some-as e exiba o resultado e seu tipo.
3. Converta uma string `"123"` para o tipo `int` e imprima o resultado.
4. Verifique se a variável `a = 10.0` é do tipo `int` ou `float` usando `isinstance()`.
5. Peça ao usuário que digite um número e mostre se ele é par ou ímpar.
6. Crie uma variável booleana com valor `True`, depois inverta seu valor e imprima.
7. Transforme o número `3.14159` em uma string e mostre o tipo da nova variável.
8. Escreva um programa que receba dois números do usuário e exiba a média entre eles.
9. Multiplique uma string por um número e mostre o resultado. Ex: `"oi" * 3`
10. Receba uma idade digitada pelo usuário e verifique se ela é maior ou igual a 18.
11. Receba uma nota (float) de um aluno e mostre se ele foi aprovado (nota >= 6).
12. Use `input()` para receber o nome do usuário e exiba uma saudação.
13. Crie três variáveis com tipos diferentes (`int`, `float`, `str`) e imprima seus valores e tipos.

## Parte 2: Funções em Python

**Objetivo:** Desenvolver a capacidade de criar, chamar e usar funções com parâmetros e retorno.

### Exercícios

1. Crie uma função que imprime "Olá, mundo!".
2. Escreva uma função que recebe um nome como parâmetro e imprime uma saudação.
3. Crie uma função que receba dois números como parâmetros e retorne a soma deles.
4. Crie uma função que retorne o quadrado de um número.
5. Faça uma função que verifique se um número é par.
6. Crie uma função que receba três notas e retorne a média.
7. Faça uma função que retorne o maior entre dois números.
8. Crie uma função que receba um texto e o retorne em letras maiúsculas.
9. Faça uma função que receba uma lista de números e retorne a soma deles.
10. Escreva uma função que retorne a quantidade de caracteres de uma string.
11. Escreva uma função que receba uma lista de nomes e imprima cada um com uma saudação.
